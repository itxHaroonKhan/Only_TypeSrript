# Learn-TypeScript-Basics

This repository contains basic TypeScript code examples to help understand the core concepts of TypeScript.

## 📚 Topics Covered

- Type Annotations (string, number, boolean, etc.)
- Interfaces and Types
- Functions with types
- Arrays and Tuples
- Enums
- Type Inference
- Type Aliases
- Classes and Objects

## 🛠 Technologies Used

- TypeScript
- Node.js (for running `.ts` files)
- VS Code (recommended editor)
